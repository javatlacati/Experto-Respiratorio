# Experto-Respiratorio
Sistema experto de ejempo para una detección simple de enfermedades respiratorias

Hasta el momento diagnostica las siguientes enfermdades:
* Asma
* Bronconeuminía
* Bronquitis
* Sars
* Gripe
* Rinofaringitis
* Sinusitis

* Congestión nasal
* Decaimiento
* Dificultades respiratorias

* Dolor de dientes
* Dolor de garganta
* Dolor en el cuerpo
* Dolor en el pecho
* Estornudos
* Falta de apetito
* Fiebre
* Flujo nasal
* Gripe
* Inflamación unos días
* Lagrimeo
* Malestar general
* Rechazo de alimentos
* Respiración agitada
* Ruidos silbantes
